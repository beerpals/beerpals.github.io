Beer Palls

Records your drinking activity since 2014.

Getting started

  1. Fork and clone beerhub repository
  2. Commit your beer git commit -m 'Mikkeller Geek Brunch Weasel' --allow-empty
  3. Push your changes git push origin master
  4. Check out your stats https://beerpals.github.io/#!/username

FAQ

How to commit beer you have drunk a few hours ago?

  • git commit -m 'St. Bernardus Abt 12' --date 'Wed Apr 9 23:53 2014 +0200' --allow-empty
